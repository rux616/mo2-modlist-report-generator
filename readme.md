MO2 Modlist Report Generator
============================
by rux616
Version: 1.0.0

This is a plugin for Mod Organizer 2 that will export your mod list (the left hand pane) to a file.

It is based on [Lazy Modlist Extractor for Mod Organizer 2](https://www.nexusmods.com/skyrimspecialedition/mods/105126) by [lazyelm](https://www.nexusmods.com/skyrimspecialedition/users/5216626) on Nexus Mods and is fully permitted by the permissions as set on the mod on the day it was accessed (2026-03-07). (See: ./assets/nexus-permissions-2026-03-07.jpg)
